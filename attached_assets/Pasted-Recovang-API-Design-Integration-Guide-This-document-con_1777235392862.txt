2. AI Integration Prompt (Technical)
Copy and paste this into your AI assistant to start building the frontend:

"I need you to build a production-ready React frontend for Recovang, a sustainability platform.

Architecture & Stack:

Use React 18 + TypeScript + Vite.
Styling: Tailwind CSS (Strictly follow the project's CSS variables).
State Management: Zustand for global auth and UI state.
Data Fetching: TanStack Query (React Query) for caching and offline fallback.
Forms: React Hook Form + Zod for validation.
Routing: React Router v6 with Protected Routes based on user roles (Collector, Agent, Brand, etc.).
Core Tasks:

Implement a Unified Registration flow (Step 1: Basic info -> Step 2: Role-specific onboarding).
Create an Axios interceptor for JWT management, handling 401 errors by attempting to call /api/v1/auth/refresh.
Implement Offline-First Submissions: Use Dexie.js (IndexedDB) to queue waste submissions when the user has no internet, and sync them automatically via the /api/v1/submissions/sync endpoint when back online.
Build a Dynamic Dashboard that adapts based on user role, pulling data from the relevant /dashboard endpoints.
Here is the endpoint list for reference: [Paste Section 1 Above]"

3. Professional UI/UX & Design Prompt
Copy and paste this to ensure the AI creates a world-class design:

"I want you to act as a Senior Product Designer and Frontend Architect. Your task is to design a 'wow-worthy' interface for Recovang that rivals top fintech apps like Revolut and Apple Card.

Design Aesthetic:

Premium Color Palette: Deep Forest Green (#1A6B3C), Metallic Gold (#D4A017), and Rich Charcoal (#1C1C2E).
Visual Style: Use Glassmorphism for cards, subtle Linear Gradients, and a spacious layout.
Typography: Strictly use Plus Jakarta Sans for headings and Lato for body text. Use JetBrains Mono for all monetary values.
Interaction Design:

Micro-animations: Use Framer Motion for page transitions, button hovers, and skeleton loaders.
Money-First UX: The user's balance and earning potential must be the most prominent visual element on the screen.
3-Tap Rule: Any critical action (submitting waste, withdrawing cash, verifying a load) must be achievable in 3 taps or less.
Standards & Quality:

Mobile-First: The app will be used primarily on mobile devices (TECNO, Infinix, etc.) in Nigeria. Optimize for touch targets and performance.
Zero Placeholders: Use your generate_image tool to create realistic user avatars, hub photos, and category icons.
Trust Signals: Use clear success/error states, verification badges, and real-time update indicators (WebSockets where applicable).
Create a design that feels like the future of waste management—clean, professional, and profitable."
