1. Authentication & Onboarding (/auth)
Method	Endpoint	Description
POST	/register	Step 1: Unified Registration for all roles
POST	/onboard/collector	Step 2: Collector-specific data onboarding
POST	/onboard/business	Step 2: Brand & Factory onboarding
POST	/onboard/agent	Step 2: Agent onboarding
POST	/login	Login for all user types
POST	/logout	Invalidate current session
POST	/verify-email	Verify account with OTP
POST	/forgot-password	Request password reset OTP
POST	/reset-password	Reset password using OTP
GET	/me	Get current authenticated user profile
POST	/refresh	Refresh access token
2. Collector Portal (/collector)
Method	Endpoint	Description
GET	/dashboard	Dashboard stats (Earnings, Weight, Streak)
GET	/profile	Get detailed collector profile
PUT	/profile	Update collector profile
POST	/bank-account	Add bank account for withdrawals
POST	/bank-account/verify	Verify bank account details
GET	/balance	Get current wallet balance
GET	/earnings/summary	Earnings by period (weekly/monthly/yearly)
GET	/earnings/history	Itemized earnings history
GET	/submissions	List waste collection submissions
GET	/submissions/:id	Specific submission details
POST	/submissions	Create new waste submission request
POST	/submissions/:id/dispute	Dispute a verification result
GET	/hubs/nearby	Find hubs within 10km radius
GET	/leaderboard	Top collectors by XP/Points
GET	/badges	List of earned milestone badges
GET	/notifications	User notifications list
PUT	/notifications/:id/read	Mark single notification as read
PUT	/notifications/read-all	Mark all notifications as read
GET	/referral	Get referral code and stats
GET	/streak	Current submission streak data
POST	/utilities/purchase	Buy airtime/data/bills using balance

7. Global Ecosystem Utilities
Submissions (/submissions)

POST / : Create new submission (multi-category)
GET /:id : Get submission detail
POST /:id/photos : Upload proof photos (Multipart/form-data)
POST /sync : Sync batch of offline submissions
Withdrawals (/withdrawals)

GET / : User withdrawal history
POST /bank : Request bank transfer
POST /airtime : Request airtime top-up
POST /bills/:billType : Pay electricity/TV/utility bills
Waste & Pricing (/waste-categories & /pricing)

GET /waste-categories/ : List categories + pricing
GET /pricing/ : Current platform standards
GET /pricing/calculate : Earnings calculator for collectors
Logistics (/logistics)

PUT /location : Update vehicle GPS
POST /pickups/request : Request hub evacuation
POST /pickups/:id/accept : Accept pickup request
System & Health (/system)

GET /health : Server health check
GET /ready : Database readiness check
